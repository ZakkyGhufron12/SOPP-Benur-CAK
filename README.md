# SOPP-Benur CAK 🦐 (AI-Enabled Monitoring)

Aplikasi mobile berbasis **React Native** yang dirancang untuk monitoring budidaya benur secara cerdas. Proyek ini mengintegrasikan **AI** untuk efisiensi operasional.

### 🚀 Key Features:
* **AI Object Detection**: Implementasi **YOLOv12** untuk deteksi objek secara real-time.
* **Full-Stack Integration**: Menggunakan **Supabase (PostgreSQL)** sebagai backend dan database.
* **Modern UI/UX**: Dibangun dengan framework **Expo** untuk performa mobile yang optimal.

### 🛠 Tech Stack:
* **Frontend**: React Native (Expo)
* **Backend**: Supabase
* **AI Model**: YOLOv12
* **Language**: TypeScript / JavaScript
